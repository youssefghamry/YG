أفضل برنامج لإدارة مكاتب المحاماة

 نظام إدارة مكاتب المحاماة والاستشارات القانونية - الإصدار الثالث - 2022

 نظام محاسبي وإداري سهل وبسيط ، متخصص في إدارة مكاتب المحاماة والإستشارات القانونية 

إدارة العملاء - إدارة القضايا - إدارة الاستشارات - المهام -  إدارة الجلسات - إدارة المدفوعات - إدارة المصروفات - إدارة المهام - إدارة التنبيهات -  إدارة المستخدين - تقارير - صفحة خاصة لكل موكل لمتابعة كل ما يتعلق بقضاياه

  • هل لديك مكتب محاماة وتبحث عن برنامج أو نظام يسهل عليك إدارة وتنظيم المكتب ؟
  • هل تبحث عن برنامج إدارة مكاتب المحاماة ؟ أو نظام إدارة مكاتب المحاماة ؟
  • هل تبحث عن أفضل برنامج لإدارة مكاتب المحاماة ؟
  • هل تبحث عن برنامج المحامي الذكي الذي يدير ويسهل عليك إدارة المكتب ؟
  • هل تعبت من كثرة الأوراق ونسيان مواعيد الجلسات والقضايا والمدفوعات ؟

خصائص عامة عن البرنامج :

  • لغة البرمجة PHP 
  • قواعد البيانات MySQL
  • البرنامج يعمل أون لاين
  • يمكن تشغيل البرنامج من أكثر من جهاز 
  • يمكن تصفحه من الموبايل
  • اضافة اللوجو والهيدر وختم المكتب
  • إضافة شروط العقد
  • طباعة العقود والسندات

خصائص ومميزات البرنامج :

قسم خاص بالعملاء :
 (اضافة اسم العميل - رقم الهاتف - الإيميل - العنوان - الجنسية - ملاحظات أخرى - مع يوزر نيم وباسوورد خاص للعميل )

أنواع العملاء : مدعي - مدعى عليه - مجني - مجني عليه وهكذا

قسم خاص بالقضايا وفيها الآتي :

  • إمكانية إضافة انواع او أصناف للقضايا
  • إمكانية إضافة درجات ومراحل التقاضي
  • أتعاب القضية
  • فرز القضايا حسب النوع أو درجة التقاضي
  • يمكن إضافة اكثر من قضية للعميل
  • ( اضافة بيانات العميل - اضافة بيانات الخصم - اضافة بيانات محامي الخصم - تصنيف القضايا - أتعاب القضايا - المحكمة التي فيها القضية - رقم القضية - درجة التقاضي - المبلغ المتفق عليه في القضية - الدفعة المقدمة - باقي المبلغ )
  • إمكانية طباعة العقد
  • إمكانية إصدار الفواتير والسندات
  • إمكانية إصدار دفعات مالية ( نظام التقسيط ) 

قسم خاص بجلسات القضية وفيه الآتي :

  •  إضافة أكثر من جلسة لكل قضية وتاريخ الجلسة وتفاصيل عن الجلسة وما حدث فيها  )
  • اظهار الجلسات في التقويم
  • امكانية طباعة جلسات بين تاريخين

قسم خاص بالاستشارات القانونية وفيها الآتي :

  • إمكانية إضافة اكثر من استشارة قانونية 
  • إمكانية اضافة استشارة سريعة او عقد استشارة لمدة معينة 
  • اضافة مبلغ الاستشارة مع خاصية الدفع على مراحل
  • قسم خاص بالمصروفات : امكانية تقسيم واضافة المصروفات 
  • قسم المستخدمين للنظام : مع امكانية اضافة مستخدم جديد وله صلاحيات خاصة

قسم المصروفات : 

  • إمكانية تقسيم المصروفات لأقسام مثل إيجار المكتب - الإنترنت - الرواتب وهكذا 
  • إمكانية إضافة المصروفات مع طباعة سند الصرف

قسم التقارير :
