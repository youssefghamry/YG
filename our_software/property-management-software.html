برنامج إدارة العقارات

  برنامج متخصص في إدارة العقارات والأملاك ( إدارة أكثر من وحدة عقارية )

خصائص عامة عن البرنامج :

  • لغة البرمجة PHP 
  • قواعد البيانات MySQL
  • البرنامج يعمل أون لاين
  • يمكن تشغيل البرنامج من أكثر من جهاز 
  • يمكن تصفحه من الموبايل
  • البرنامج باللغة العربية والإنجليزية
  • إمكانية عمل نسخة احتياطية من قاعدة البيانات
  • اضافة اللوجو والهيدر وختم الشركة.

خصائص ومميزات البرنامج :

إضافة بيانات ملاك العقارات :

  • إمكانية اضافة مالك العقار ( اسم مالك العقار - الهاتف - العنوان - الإيميل - رقم الـ ID - شركة أم فرد - رقم السجل التجاري - صندوق البريد - الرمز البريدي - مرفقات من صور وغيره - يمكن إضافة أكثر من مالك 

إضافة بيانات المستأجرين  :

  • إمكانية إضافة أكثر من مستأجر ( الاسم - الهاتف - العنوان - الإيميل - رقم الـ ID - شركة أم فرد - رقم السجل التجاري - صندوق البريد - الرمز البريدي - مرفقات من صور وغيره 

إضافة تفاصيل البنايات والوحدات العقارية :

  • إمكانية اضافة أنواع الوحدات العقارية مثل :  شقة - بناية - محل - مكتب - معرض … )
  • إمكانية اضافة خصائص الوحدات العقارية مثل : غرفة نوم - حمام - مطبخ - صالة ….
  • إمكانية تقسيم وإضافة الحالات للوحدات العقارية مثل : شاغر - معلق - تم التأجير …
  • إمكانية اضافة وحدات عقارية جديدة مع امكانية تقسيم الوحدة العقارية لأدوار مثل الدور الأرضي الدور الأول الدور الثاني وهكذا وبداخل كل دور يمكن التقسيم وإضافة الوحدات مثل محلات أو شقق أو غيره بحيث يظهر في كل دور عدد المحلات أو الشقق الموجودة 
  • في داخل كل وحدة يمكن اضافة خصائص كل وحدة مثلاً : يوجد شقة ويوجد بها 2 غرفة نوم + مطبخ + صالة أو مجلس + 2 حمام 
  • ايضا اضافة رقم عداد الكهرباء ورقم عدد المياة في تفاصيل الوحدة العقارية 
  • بعد ذلك يمكن تغيير حالة الوحدة العقارية ( شاغرة - معلقة - تم التأجير )
  • أيضا يمكن تأجير الشقة بكل سهولة وفيها يتم استدعاء بيانات مالك العقار وكذلك بيانات المستأجر 

إنشاء العقود والدفعات :

  • إمكانية عمل عقد بين المالك والمستأجر مع إمكانية طباعة العقد 
  • عند إنشاء عقد جديد فيه إمكانية لإضافة مبلغ الإيجار الشهري وعدد الشهور ليتم بعد ذلك إنشاء عدد الشهور التي تم استئجارها وفي كل شهر مبلغ التأجير 
  • إمكانية دفع شهر الإيجار على اكثر من دفعة 
  • امكانية طباعة سند قبض للمستأجر أو ارساله على الواتساب

قسم خاص بالمصروفات : 

  • إمكانية تقسيم المصروفات مثل مصروفات صيانة - مصروفات إدارية - مصروفات أخرى ..
  • إمكانية اضافة مصروف مع الحاق المصروف بوحدة عقارية معينة بحيث يتم معرفة كم صرفت هذه الوحدة العقارية 

قسم خاص بالمستخدمين للنظام : 

  • إمكانية إنشاء أكثر من مستخدم وكل مستخدم له صلاحية محددة يحددها الأدمن

قسم خاص بإعدادات النظام : 

  • إمكانية اضافة اللوجو الخاص بالشركة
  • إمكانية اضافة هيدر الفواتير والسندات والعقود
  • إمكانية اضافة الختم الخاص بالشركة 
  • إمكانية حفظ نسخة احتياطية من قاعدة البيانات 

قسم خاص بالتقارير وفيه الآتي  : 

  • إمكانية الاطلاع على التقارير الخاصة بالنظام بالكامل بين تاريخين مع امكانية الطباعة أو تصدير ملف pdf او ملف اكسيل أو النسخ 
  • الاطلاع على تقارير العقود 
  • الاطلاع على تقارير المدفوعات
  • الاطلاع على تقارير المصروفات
  • امكانية معرفة الوحدات العقارية من خلال حالتها ( شاغرة - معلقة - مستأجرة ) لكل البنايات أو لكل بناية على حدة
  • إمكانية الاطلاع على الدفعات الخاصة بكل مستأخرة ومعرفة المبلغ المدفوع والمبلغ المتبقي

أهم خصائص ومميزات برنامج إدارة العقارات والأملاك

نظام سحابي آمن

نظام سحابي آمن ( كلاود ) يمكن تشغيل وإدارة النظام من أكثر من جهاز ومن أكثر من مكان في نفس الوقت - ويمكن التصفح من خلال الهاتف أو التاب

البحث السريع

إمكانية البحث السريع عن اي مستأجر من خلال رقم الهاتف أو اسم المستأجر وكذلك البحث عن أي مالك بناية أو وحدة عقارية في النظام

تقارير كاملة

إمكانية الاطلاع على تقارير خاصة بين تاريخين ويمكن الاطلاع على تقارير مفرده أو كلية لكل ما في النظام بين تاريخين مع تصدير لملف اكسيل

النظام قابل للتطوير والتعديل

من المميزات التي نتميز بها عن غيرنا إمكانية التعديل والتطوير في النظام حسب رغبة العميل ، إيماناً منا باختلاف متطلبات العملاء في النظام

إدارة المستخدمين والصلاحيات

إمكانية اضافة عدد لا محدود من المستخدمين ومدراء النظام مع إضافة الصلاحيات الخاصة لكل مستخدم أو مجموعة مستخدمين يحددها الأدمن

إدارة المستأجرين

إمكانية إضافة المستأجرين ( اسم المستأجر - رقم الهاتف - العنوان - الجنسية - رقم الهوية ) ويمكن البحث عن اي مستأجر من خلال الاسم ورقم الهاتف والإيميل

إدارة ملاك البنايات

إمكانية إضافة أكثر من مالك ( اسم المالك - رقم الهاتف - العنوان - البناية التي يمتلكها ) ويمكن البحث عن اي مالك بناية من خلال الاسم ورقم الهاتف والإيميل ويمكن إعطائه صلاحيات محددة

إدارة الوحدات العقارية

إمكانية إضافة أكثر من بناية مع تفاصيلها ويمكن تقسيمها لطوابق ووحدات عقارية مثل ( محل - شقة - مكتب - معرض )

إدارة العقود

إمكانية إضافة أكثر من عقد في النظام ويمكن طباعة العقد وتحديد الإيجار الشهري ومدة العقد

إدارة المدفوعات المالية

إمكانية عمل أكثر من دفعة للعقد الواحد و تحصيل الإيجار الشهري مع طباعة سند القبض

إدارة مواعيد المدفوعات

إمكانية إضافة أكثر من موعد للمدفوعات مثل مواعيد ( شهرية - سنوية - نصف سنوية - ربع سنوية ) مع تحديد مواعيد المدفوعات

إدارة المصروفات

قسم كامل خاص بإدارة المصروفات مع امكانية تقسيم المصروفات لأقسام مثل ( ايجارات - رواتب - مصاريف إدارية - مصاريف صيانة وغيرها ) مع طباعة سند الصرف مع التقارير

دعم فني مستمر 7/24 / 365

نقدم الدعم الفني والتقني الكامل والمستمر على مدار الساعة والاستجابة السريعة من خلال البريد الإلكتروني أو الاتصال هاتفيا أو التواصل مباشرة عبر الواتساب

خادم خاص مع الحماية والأمان

نوفر لكم خادم سحابي خاص مع قاعدة بيانات خاصة مما يضمن لكم السرية والخصوصية والأمان لكافة البيانات الخاصة بكم مع النسخ الاحتياطي

دعم ضريبة القيمة المضافة

النظام يدعم ضريبة القيمة المضافة ويمكن التحكم في نسبة الضريبة ويمكن معرفة تقارير شاملة عن قيمة الضريبة بين تاريخين مما يساعد الورشة في حساب الضريبة

دعم الفاتورة الالكترونية QR Code

النظام يدعم الفاتورة الالكترونية ونظام ال QR Code في الفواتير وعروض الأسعار والسندات مع إصدار الفواتير كملف pdf أو إمكانية الطباعة
